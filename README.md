# AI 简历筛选 Demo

一个本地运行的 AI 招聘辅助工具：
- 批量读取候选人简历
- 可选匹配候选人面试视频做辅助分析
- 输出结构化评分结果，帮助快速筛选高潜人才

## 代码用途
本项目用于演示“本地大模型 + 多模态辅助”的招聘评估流程。

系统会基于岗位要求，对候选人进行三层评分：
- Layer 1（硬技能）
- Layer 2（软技能）
- Layer 3（潜力）

最终按固定权重计算综合得分，用于批量对比候选人。

## 核心功能
- 批量上传简历：支持 `txt/md/pdf/docx`
- 批量上传视频：支持 `mp4/mov/mkv/avi/webm`
- 姓名自动匹配：根据文件名和简历内容匹配同一候选人的视频
- 本地 LLM 评分：通过 `llama-cpp-python` 加载本地 GGUF 模型（默认 Qwen）
- 视频辅助信息：
  - `MediaPipe FaceMesh` 提取基础行为统计
  - `SenseVoice`（通过 FunASR）进行语音转写（依赖 ffmpeg）
- 结果展示：表格汇总 + 候选人明细（分层得分、综合得分、风险点、匹配依据）

## 快速启动
1. 直接双击运行 `one_click_start.bat`（推荐）
2. 脚本会自动：
   - 检查/准备 Python 环境
   - 创建虚拟环境并安装依赖
   - 启动 Streamlit 页面
3. 浏览器打开后，上传简历与岗位要求即可开始分析

## 运行前准备
- 本地 GGUF 模型文件（默认放在 `models/` 目录）
- 建议安装 ffmpeg（用于视频语音转写）

## 说明
- 项目为本地演示工具，评估结果用于辅助决策，不应替代最终人工面试判断。
